# ssk0

See on omanäoline veebiprojekt, mis on loodud Bootstrapi baasil, kuid kohandatud unikaalse disainiga.

## Live Demo

Vaata projekti töös: [https://002ssk.github.io/ssk0/](https://002ssk.github.io/ssk0/)

## Struktuur

- `index.html` - Pealeht
- `css/style.css` - Stiilid
- `bootstrap/` - Algne Bootstrap versioon
- `CSS/`, `google.projektt/`, `veebileht/` - Teised projektid

## Omanäolised elemendid

- Gradient taustad ja värvid
- CSS animatsioonid (fade-in, pulse, float)
- Interaktiivsed nupud ja hover-efektid
- Unikaalne logo ja kujundus
